Spirographs

My senior year of high school, I found spirographs to be extremely fascinating. I creating a program to create randomly generated spirographs based on some mathematical formulas, which I have recreated here using P5.js for image rendering. I still find myself getting lost in these pictures.

Click anywhere in the canvas to randomly generate a new spirograph.
