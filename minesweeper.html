Minesweeper

My implementation of the classic game Minesweeper using P5.js for image rendering and vanilla JavaScript for logic. This version has several preset board sizes, game difficulties, and variable tile size.

CONTROLS:
left-click: uncover tiles and any surronding empty or numbered tiles
shift + left-click: flag or unflag the tile clicked on
p: pause or unpause the game
